The components of the numpy array can be extracted by providing a character corresponding to the axis as the first argument: >>> p.get('x') 3.0 >>> p.get('y') 5.0 >>> p.get('z') 2.0

Now, let's test the representation >>> p.get() [3.0, 5.0, 2.0]
