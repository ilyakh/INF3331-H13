save(self, path=None)

source code 

Compiles the body of this Element into a .scad file. If the path is omitted, the path will default to a string containing the current timestamp in format:

YEAR-MONTH-DAY-HOUR-MINUTE-SECOND.scad

The default timestamps are of second resolution, so two saves made during the same second of time will only yield the most recent one.

Parameters:
  • path (string) - path incl. filename to compile the .scad code to. must be a valid and writable location in the filesystem.
